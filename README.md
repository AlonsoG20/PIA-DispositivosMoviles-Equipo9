# PIA-DispositivosMoviles-Equipo9
Sistema de control de flotilla con geofencing que permite al chofer registrar paradas, al despachador asignar rutas y al gerente supervisar la operación. Genera reportes de paradas no autorizadas fuera del perímetro y consumo de combustible para mejorar el control y la eficiencia operativa.
